# _This project has been created as part of the 42 curriculum by ncarrera and gubusque._

## This project emulates the core functionalities of a shell (Like bash), implementing the standard functionality of such a shell,
## and allowing you to customize aspects of it to your liking.

### To run this minishell, simply clone the repository, open a terminal in the directory where you copied this repo, and type "make".
### You can also run "make reclean", which will remake the program, and delete the non-executable/essential files, leaving a clean directory. 

#### [This article](https://brennan.io/2015/01/16/write-a-shell-in-c/) does a great job at explaining how a shell starts up, what processes
#### it runs, what it does, and shows you how you could implement a shell on your own while you learn all about the things that make it tick.
